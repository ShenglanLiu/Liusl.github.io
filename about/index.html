个人简介：

刘胜蓝，Human Perception Computing（HPC）实验室负责人，大连理工大学副教授。

研究生及个人科研工作：主要研究方向为以人为中心的感知计算（包括计算机视觉、听觉），机器学习以及数据可视化等领域。博士毕业论文获辽宁省优秀博士学位论文。目前，已发表论文50余篇，其中，在AAAI等顶级会议以及IEEE汇刊、数据领域顶级期刊Scientific Data等高水平期刊（IEEE Trans. On NNLS等）以第一作者及通讯作者发表论文二十余篇。获大连市青年科技之星，累计获得辽宁省学术成果奖学术成果奖6项。主持国家自然科学基金一项，CCF-百度松果基金一项，东软集团软件架构国家重点实验室开放课题一项，博士后面上资助一项，以及多项企业合作横向课题。

目前，担任SCI检索期刊Neurocomputing（中科院TOP期刊）的编委。在WCICA2016及CASE2017（与ICRA和IORS作为IEEE RAS下的三个重要会议）、ICONIP2017、ICONIP2020（CCF C类会议）本领域学术会议上受邀组织相关的Special Session。同时，本人为CVPR、ICML等顶级会议的审稿人，以及IEEE Transactions on Pattern Analysis and Machine Intelligence、IEEE Transactions on Neural Networks and Learning Systems、IEEE Transactions on Cybernetics、ACM Computing Surveys、Information Sciences、Neural Networks等10余个SCI检索期刊的审稿人。

社会兼职：中国自动化学会混合智能专业委员会委员；新工科联盟智能驾驶协同育人工作委员会委员

交流合作：目前，实验室与新加坡国立大学、中科院北京自动化研究所、清华大学、华南理工大学等多所高校均有深入合作。

研究生毕业去向：

2020年（第一届毕业生）：刘相（阿里），王洋（今日头条）

本科生教学、竞赛及科研：人工智能创新实践班成立于2017年9月，目前有授课教师两名：刘胜蓝，王飞龙；已有本科生400余人（截止2018年9月），主讲课程为：

  1. 机器学习理论与应用（刘胜蓝）
  2. 深度学习基础（计算机视觉、听觉等）（刘胜蓝）
  3. 数据科学导论（刘胜蓝）
  4. 智能算法的FPGA实现 （王飞龙）
  5. 智能机器人与人工智能（王飞龙）
  6. Python、MATLAB基础及机器学习实现（HPC团队）

人工智能创新实践班是创新创业学院为促进工业4.0、人工智能2.0发展，培养国家新工科人才的重要举措。实践班结合机器人、物联网、VR技术主要开展三个方向的创新活动及项目研究：1. 机器学习算法及应用 2. FPGA在机器视觉上的开发；3. 人工智能在智能机器人上的应用。实践班引入算法——软件——硬件联动的“三位一体”培养模式；同时，将人工智能+、本-硕-博立体培养方案作为人工智能实践班的特色，促进人工智能理论和方法的落地，增强学生理论联系实践的能力，培养一批会思考、能动手的综合能力人才。在此基础上，以项目促进学生创业，联合创新创业学院的创客基地，开展学生人工智能创业实战，努力实现学生创新创业融合，形成“大众创业、万众创新”的新局面。

目前，作为人工智能团队的主要骨干，指导硕士、博士的相关科研工作，指导人工智能实践班本科生竞赛和发表论文。目前，获国家级教改项目一项；实践班成立之初，在2017年10月的第二届中国机器人及人工智能大赛作为指导教师获得2个一等奖的好成绩，并被大连理工大学新闻网、今日头条等多家媒体报道。
